# 📊 DS-ML Bootcamp

> **Interactive guide** with cyberpunk shell, sound engine, markdown editor, synthwave animations, and dark/light themes.
> **🌐 Live:** [enlyl.github.io/ds-ml-bootcamp](https://enlyl.github.io/ds-ml-bootcamp)

---

## English

Welcome to the **Data Science & Machine Learning Bootcamp** repository!

This repository contains all lessons, code examples, and resources used during a one-month intensive bootcamp. The program is designed to take participants from **zero to practical projects**, covering the full ML workflow:

1. Collect Data
2. Preprocess Data
3. Split into Train/Test
4. Choose Model
5. Train Model
6. Evaluate Model
7. Deploy Model

### 🎯 Goal

Our mission is simple: go from **unreal to real**, from **unimaginable to imaginable** — practicing the entire DS/ML path, step by step, **in just one month**.

### 📂 Structure

```
ds-ml-bootcamp/
├── index.html          # Interactive guide (PWA, 9 themes, i18n)
├── lessons/            # Markdown lessons (00–04)
├── assignments/        # Assignment descriptions + notebooks
├── code/               # Python scripts + Jupyter notebooks
├── dataset/            # Raw datasets (houses, cars)
└── submissions/        # Student work
```

### 🚀 Quick Start

```bash
git clone https://github.com/Enlyl/ds-ml-bootcamp.git
cd ds-ml-bootcamp
npx http-server -p 3000
# open http://localhost:3000
```

Or open `index.html` directly in a browser.

### ✨ Features

- **9 themes** — Cyber, Retro, Gaming, Hacker, Synthwave, Surreal, Minimal, Minimal Light, Light
- **Bilingual** — English (default) / Russian toggle
- **Sound engine** — ambient, key clicks, modem handshake, sub-bass
- **Markdown reader** — open `.md` files with syntax highlighting
- **Colab buttons** — open Jupyter notebooks directly in Google Colab
- **PWA** — installable as a standalone app
- **Keyboard nav** — arrows, 1–9, `/` search, `Ctrl+T` theme cycle
- **Search** — filter steps by title/description
- **Back to top** — smooth scroll button

## Русский

Добро пожаловать в репозиторий **Data Science & Machine Learning Bootcamp**!

Этот репозиторий содержит все уроки, примеры кода и ресурсы, используемые во время месячного интенсивного bootcamp. Программа разработана, чтобы провести участников от **нуля до практических проектов**, охватывая полный ML workflow:

1. Сбор данных
2. Предобработка данных
3. Разделение на обучающую и тестовую выборки
4. Выбор модели
5. Обучение модели
6. Оценка модели
7. Развёртывание модели

### 🎯 Цель

Наша миссия проста: перейти от **нереального к реальному**, от **невообразимого к вообразимому** — практикуя весь DS/ML путь, шаг за шагом, **всего за один месяц**.
